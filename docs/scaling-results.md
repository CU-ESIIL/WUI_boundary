# Scaling results

Once the boundary object has been defined and measurement scale has been made explicit, the next step is to examine how the reported perimeter changes across a range of scales.

The figure on this page summarizes the relationship between measurement scale and measured boundary length for the current demonstration analysis. Each point represents the length obtained when the same boundary object is traced using a different effective ruler length.

The resulting pattern illustrates a central property of complex spatial boundaries. As the measurement scale becomes finer, the measured perimeter increases. Coarser measurement smooths over many small irregularities, while finer measurement begins to follow the detailed geometry of the boundary.

In the current demonstration the boundary geometry is synthetic, designed only to illustrate the measurement logic. The purpose of the figure is therefore conceptual rather than empirical. It shows how boundary length can behave as a scale-dependent quantity rather than a fixed attribute.

In future analyses this framework can be applied to empirical WUI datasets to examine how real landscapes express the same scaling relationship. The goal is not merely to compute a single perimeter value, but to understand how that value emerges from the interaction between boundary definition and measurement scale.
