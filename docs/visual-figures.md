# Visual figures

Some figures in this project are generated directly through the analytical pipeline, while others are conceptual images designed to clarify the geometry, measurement logic, or remote-sensing implications of the argument. The repository can be prepared to display those figures even when the binary image assets are added manually at a later stage.

This page therefore serves two purposes. It collects the visual figures already generated through code, and it provides stable locations where additional conceptual or interpretive figures can be placed once they are uploaded to the repository. In that sense it is both a gallery and a placeholder for a more visually complete version of the manuscript.
