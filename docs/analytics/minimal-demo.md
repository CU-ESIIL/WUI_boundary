## Rerun instructions

You can regenerate these outputs either locally or in GitHub Actions.

### GitHub Actions

1. Open **Actions** in this repository.
2. Run **Boundary Scaling Demo** with **workflow_dispatch**.
3. Download artifact: `boundary-scaling-minimal-demo` for full `outputs/minimal_demo/` files.

### Local run

```bash
pip install -r requirements.txt
python -m unittest discover -s tests -v
python scripts/run_minimal_boundary_scaling.py
```

Generated files:

- `outputs/minimal_demo/boundary_scaling_plot.png`
- `outputs/minimal_demo/boundary_scaling_summary.csv`
- `outputs/minimal_demo/run_summary.md`
