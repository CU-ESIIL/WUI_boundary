## What is new in Draft 2

- **Stronger visual hierarchy** to support a clearer narrative flow across controls, map, scaling, and analysis prompts.
- **Landscape archetypes** so users can switch among exurban fringe, suburban edge, and mountain mosaic profiles.
- **Sensitivity comparison panel** to show definition sensitivity and measurement sensitivity side by side.
- **Tighter comparative framing** around two different operations:
  - changing the map (definition choices alter the boundary object)
  - changing the ruler (measurement scale alters traced length from that object)

## Scope note

- This app remains self-contained and synthetic for communication and design iteration.
- It does not yet ingest real geospatial datasets or run the repository's future production analytics pipeline.

See [Draft 1 — WUI Boundary Story Lab](draft-1.md), [Roadmap to Real Data](roadmap-to-real-data.md), and [Boundary Analytics Scaffold](../analytics/scaffold.md) for continuity between conceptual UI drafts and implementation planning.
